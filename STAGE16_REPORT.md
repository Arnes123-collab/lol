# Stage 16 - Финальный модуль: личный кабинет клиента

Добавлено:

- `/admin/client-portal` - управление доступами клиентов.
- `/client/login` - вход клиента по телефону и коду.
- `/client/dashboard` - личный кабинет клиента.
- Клиент видит тариф, статус, сроки, комментарий тренера и тренировки.
- Из кабинета есть переход на отправку отчёта после тренировки.
- Добавлена ссылка "Кабинет" в основное меню сайта.
- Добавлена SQL-заготовка `sql/stage16_client_portal.sql`.

Локальная версия сохраняет кабинет клиента в браузере через localStorage.
Для домена нужно подключить таблицу `client_portal_access` в Supabase.

Демо-вход клиента:

- Телефон: `+7 700 000 00 01`
- Код: `1111`

Админка:

- `/admin/client-portal`

Клиент:

- `/client/login`

ЭТАП 13 - УПРАВЛЕНИЕ СОДЕРЖИМЫМ САЙТА ЧЕРЕЗ АДМИНКУ

1. Запусти сайт:
   npm install --legacy-peer-deps
   npm run dev:network

2. Открой админку:
   http://localhost:3000/admin/login

3. Пароль:
   admin123

4. Зайди:
   http://localhost:3000/admin/site-settings

5. Что можно менять:
   - имя тренера
   - бренд
   - стаж
   - телефон
   - WhatsApp
   - Instagram
   - город/страна
   - главный заголовок
   - текст главной страницы
   - биография
   - подпись тренера
   - тарифы

6. После сохранения открой или обнови главную страницу:
   http://localhost:3000

ВАЖНО:
В локальной версии данные сохраняются в браузере через localStorage.
Для настоящего сайта на Vercel нужно подключить Supabase-таблицу site_settings.

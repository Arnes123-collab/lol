# Как открыть сайт на компьютере

1. Распакуй ZIP.
2. Открой папку `agashka-powerlifting-local`.
3. В адресной строке проводника напиши `cmd` и нажми Enter.
4. Выполни:

```powershell
npm install
npm run dev
```

5. Открой в браузере:

```text
http://localhost:3000
```

Админка:

```text
http://localhost:3000/admin/login
```

Пароль:

```text
admin123
```

Если пишет `next не является командой`, значит не выполнен `npm install`.

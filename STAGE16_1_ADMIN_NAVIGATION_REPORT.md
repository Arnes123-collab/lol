# Stage 16.1 - Professional Admin Navigation

Добавлена профессиональная верхняя навигация админ-панели.

## Что добавлено

- Фиксированная верхняя панель во всех разделах `/admin/*`.
- Кнопка `← Назад`.
- Кнопка `Вперёд →`.
- Быстрый переход в главную админ-панель.
- Быстрые разделы:
  - Панель
  - Клиенты
  - Конструктор
  - Упражнения
  - Сайт CMS
  - Бренд
  - Отзывы
  - Кабинет клиента
  - Система
- Кнопка `🌐 Сайт` - сразу открывает основной сайт `/`.
- Кнопка `🚪 Выход` - выход из админки.
- Мобильная горизонтальная навигация с прокруткой.
- Активный раздел подсвечивается красным.
- Аватар тренера сдвинут ниже, чтобы не перекрывать верхнюю панель.

## Новый файл

`src/components/admin/navigation/AdminTopNavigation.tsx`

## Изменённые файлы

- `src/app/admin/layout.tsx`
- `src/components/admin/AdminAvatar.tsx`
- `src/app/globals.css`

## Проверка

Открыть любой раздел админки:

- `/admin/site-settings`
- `/admin/exercises`
- `/admin/plan-builder`

Верхняя панель должна быть доступна сразу, без прокрутки вниз.

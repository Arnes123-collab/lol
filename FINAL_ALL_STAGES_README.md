# AGASHKA POWERLIFTING - финальная объединённая версия

Эта версия собрана на базе Stage12 и включает все предыдущие этапы:

1. Базовый сайт
2. Supabase-заготовки
3. Клиенты и заявки
4. Тренировочные планы
5. Начальный планировщик
6. PDF / WhatsApp / печатный вид
7. Оплаты и статусы клиентов
8. Отчёты клиента после тренировки
9. Подготовка к запуску и защита админки
10. Конструктор тренировочных планов
11. Exercise Library CMS
12. Светлый редизайн и личный бренд тренера

## Локальный запуск

```powershell
npm install
npm run dev
```

Открыть:

http://localhost:3000

Админка:

http://localhost:3000/admin/login

Демо-пароль:

admin123

## Главные разделы

- /admin - панель управления
- /admin/clients - клиенты
- /admin/plan-builder - конструктор тренировочных планов
- /admin/plan-builder/saved - сохранённые планы
- /admin/exercises - база упражнений
- /admin/payments - оплаты и статусы
- /admin/reports - отчёты клиентов
- /admin/brand - личный бренд и фото тренера
- /about - о тренере
- /plans/demo/view - пример просмотра плана

## Важно

Это локальная/демо-версия. Для настоящей интернет-ссылки проект надо развернуть на Vercel и подключить Supabase.

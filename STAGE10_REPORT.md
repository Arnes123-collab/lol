# Этап 10 - Админ-конструктор тренировочных планов

Сделано:
- Добавлен раздел `/admin/plan-builder`.
- Добавлен раздел `/admin/plan-builder/saved`.
- Добавлена библиотека упражнений с изображениями: жим, присед, тяга, тяга в наклоне, Face Pull и другие.
- В конструкторе можно выбрать клиента, телефон, цель, тариф, срок, 3/4/5 тренировочных дней.
- В план можно добавить упражнение, подходы, повторения, вес, процент, отдых и комментарий тренера.
- План можно сохранить в браузере, распечатать, скачать PDF и скопировать для WhatsApp.
- Обновлены тарифы:
  1. Бесплатное занятие - 0 тенге.
  2. План на месяц - 6 000 тенге / 15 $.
  3. План на 12 недель - 15 000 тенге / 35 $.
  4. План на 16 недель - 18 000 тенге / 40 $ - подготовка к соревнованиям.
  5. Полное онлайн ведение - 40 000 тенге / 85 $ в месяц.
- Обновлён блок тарифов на лендинге.
- В админку добавлена кнопка «Конструктор».

Важно:
Сейчас сохранение конструктора работает в localStorage браузера. Для боевого режима следующий этап - перенос сохранения планов в Supabase, чтобы планы не терялись при смене браузера или компьютера.

Проверка:
- `npx tsc --noEmit` прошёл без ошибок.
- `npm run build` начал production-сборку, но в песочнице не завершился по времени. На компьютере или Vercel сборку нужно повторить.

# Этап 8 - отчёты клиента после тренировки

## Что добавлено

### Публичная форма клиента
Маршрут: `/reports/new`

Клиент может отправить:
- выполнил / не выполнил / частично
- дату тренировки
- название программы и день тренировки
- рабочие веса по упражнениям
- подходы, повторения, RPE
- вес тела
- самочувствие 1-10
- энергия 1-10
- боль 1-10
- зона боли
- комментарий
- вопрос тренеру
- ссылку на видео техники

### Админ-раздел отчётов
Маршрут: `/admin/reports`

Тренер видит:
- все отчёты клиентов
- количество отчётов
- сколько выполнили
- сколько частично выполнили
- сколько не выполнили
- сколько отчётов с болью 7+
- средний уровень боли
- фильтр по статусу
- фильтр “боль 7+”
- поиск по клиенту, программе и дню
- рабочие веса в таблице
- комментарии и вопросы клиента
- ссылку на видео техники
- печатный отчёт
- копирование отчёта для WhatsApp

### Демо-режим
Пока нет личного кабинета и Supabase-подключения для отчётов, форма сохраняет новые отчёты в `localStorage` браузера.
Это правильно для первого варианта 8-го этапа: можно проверить сценарий без сложной авторизации клиента.

## Как пользоваться
1. Тренер даёт клиенту ссылку `/reports/new`.
2. Клиент после тренировки заполняет форму.
3. Клиент нажимает “Сохранить отчёт” или “Скопировать для WhatsApp”.
4. Тренер открывает `/admin/reports` и видит отчёты.
5. Если боль 7+ - тренер сразу корректирует нагрузку.

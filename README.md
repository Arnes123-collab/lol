# agashka.powerlifting - исправленная версия проекта

Это чистая версия сайта без `node_modules` и `.next`.

## Что входит

1. Лендинг тренера - главная страница, блоки услуг, цены, результаты, форма заявки.
2. Форма заявки клиента - отправляет данные в Supabase.
3. Админ-вход - `/admin/login`.
4. Панель тренера - статистика заявок и переходы к клиентам/планам.
5. Модуль тренировочных планов - список планов, создание плана, редактор упражнений по дням.

## Запуск локально

```bash
npm install
cp .env.example .env.local
npm run dev
```

Открыть в браузере: `http://localhost:3000`

## Переменные окружения

```env
NEXT_PUBLIC_SUPABASE_URL=ваш_url_supabase
NEXT_PUBLIC_SUPABASE_ANON_KEY=ваш_anon_key
ADMIN_PASSWORD=ваш_пароль_для_админки
```

Если `ADMIN_PASSWORD` не задан, демо-пароль: `admin123`.

## Supabase

В Supabase откройте SQL Editor и выполните файл:

```text
supabase_schema.sql
```

Он создаст таблицы:

- `clients`
- `exercises`
- `training_plans`
- `training_plan_items`

И добавит базовый справочник упражнений.

## Реализованные этапы 1-12

Проект включает лендинг, заявки, админ-панель, конструктор тренировочных планов, PDF/WhatsApp/печать, оплаты, отчёты клиентов, базу упражнений Exercise CMS, светлый редизайн и раздел личного бренда тренера.

Актуальные тарифы:

1. Бесплатное занятие - 0 тг.
2. План на месяц - 6 000 тг / 15 $.
3. План на 12 недель - 15 000 тг / 35 $.
4. План на 16 недель - 18 000 тг / 40 $.
5. Полное онлайн ведение - 40 000 тг / 85 $ в месяц.

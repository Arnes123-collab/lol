## Важно

В демо-версии фотографии сохраняются в localStorage браузера. Это удобно для локального просмотра. Для боевого сайта нужно подключить Supabase Storage bucket `trainer-brand-media` и таблицы из файла `sql/stage12_trainer_brand.sql`.

## Проверить после запуска

- / - главная страница в светлом дизайне.
- /about - страница о тренере.
- /admin/brand - управление личным брендом.
- /admin - аватар тренера и ссылка на бренд.
- /admin/plan-builder - конструктор не должен быть сломан.
- /admin/exercises - база упражнений не должна быть сломана.
